## 主要实现方式
* 完成这个app主要是为了练习tableview的运用
* 项目主要是基于tableview实现，使用第三方移动端数据库realm实现数据的增删改查以及排序的功能
* 项目涉及到了页面的出栈和压栈功能，正向传值和反向传值等操作
